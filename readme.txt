Flexible tag-based image grid and slider with frames, mats, overlays, and shortcode overrides.

== Description ==

Tagged Media Gallery is a flexible, tag-based image gallery for WordPress that lets you display Media Library images as either a square grid or a responsive slider — without creating separate galleries.

Built originally for a nonprofit art gallery, this plugin focuses on clean presentation, flexibility, and professional framing options.

Instead of managing galleries manually, simply tag your images and display them anywhere using a shortcode.

= Key Features =

• Display images by Media Tag taxonomy  
• Square grid layout (fully responsive)  
• Flexible slider with fixed height (no layout shifting)  
• Frame and mat styling options  
• Custom frame & mat color pickers  
• Overlay or Below text positioning  
• Hover or Always-on overlay modes  
• Mobile-specific text display options  
• Lightbox with autoplay pause  
• Keyboard navigation + swipe support  
• Per-shortcode overrides (override global settings per gallery)

= How It Works =

Tagged Media Gallery uses a custom Media Tag taxonomy.

1. Go to Media → Library  
2. Click an image  
3. Add a Media Tag (example: featured, events, portraits)  
4. Use the shortcode:

[tagged_gallery tags="featured"]

Create unlimited galleries simply by using different tags.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin
3. Use shortcode:

[tagged_gallery tags="your-tag"]

== Frequently Asked Questions ==

= Can I override settings per gallery? =

Yes. Example:

[tagged_gallery type="slider" frame="false" nav="true"]
