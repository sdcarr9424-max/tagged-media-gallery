# WHS Tag-Based Media Gallery

A flexible tag-based image grid and slider for WordPress.

---

## 💛 Built for a Nonprofit

WHS Tag-Based Media Gallery was created for the Gateway Art Gallery to provide flexible, professional image presentation without expensive gallery systems.

---

## Features

- Grid or Slider display
- Frame + Mat styling
- Custom color pickers
- Overlay or Below text
- Hover or Always-on overlay
- Lightbox
- Keyboard + swipe navigation
- Per-shortcode overrides

## Usage

Tag images in the Media Library using **Media Tags**.

Display with:

[tagged_gallery tags="featured"]


Override settings:


[tagged_gallery type="slider" frame="false" overlay_always="true"]
